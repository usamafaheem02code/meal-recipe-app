A simple, responsive meal recipe web application built using **HTML**, **Tailwind CSS**, and **JavaScript**. It fetches data from [TheMealDB API](https://www.themealdb.com/api.php) and displays detailed recipes.

## Features
- Search meals
- View meal details
- Responsive design
- Loader animation

## Technologies
- HTML
- Tailwind CSS
- JavaScript (ES6)
- Fetch API
